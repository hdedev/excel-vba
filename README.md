# Excel - VBA Scripts

## Giới thiệu
Tập hợp các hàm và tiện ích Excel VBA hỗ trợ nghiệp vụ kế toán, giúp tự động hóa xử lý dữ liệu và nâng cao hiệu quả làm việc.


### [NumberConvert.bas](docs/NumberConvert.md)

Module VBA chứa 2 hàm chuyển đổi số thành chữ:  
- Đọc số thành chữ tiếng Việt  
- Đọc số thành chữ tiếng Anh  

Dùng cho Excel để hiển thị giá trị tiền hoặc số liệu dưới dạng chữ.
